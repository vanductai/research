3.  **Drafting**: Write Parts I-V using Side A/B labels.
4.  **Reveal**: **NOW** open `_blind_mapping.md`, reveal sides.
5.  **Final Summary**: Write Part VI Verdict + Part VII SYNTHESIS.

    -   **C. Open Questions**

### STEP 10: NOTIFY USER

- **Status**: Pipeline complete. User reads output.
