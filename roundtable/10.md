# Debate Arena v2.0 - Overview

**Multi-agent debate competition - Claude Cowork - Designed by EZLAW**  
*Stress-test ý tưởng phức tạp qua tranh luận có cấu trúc — không phải 1-shot answer*

## Nhiệm vụ & Mục đích

### Mục đích
Chuyển từ 1-shot LLM answer (collapse vào prior của model) sang quy trình tranh luận có cấu trúc — **force epistemic diversity, adversarial testing, fact verification, và synthesis** để sinh insight cao hơn một câu trả lời đơn lẻ.

### Cách dùng
User gõ topic trực tiếp vào Cowork → pipeline tự chạy 10 agents qua 4 vòng tranh luận với persona diversity và ensemble judging → output verdict + Synthesis (Reconciliation / Residual Disagreement / Open Questions / Actionable Recommendations) cho topics chính sách, đạo đức, công nghệ, xã hội.

## Công nghệ

- **PLATFORM**: Claude Cowork (Anthropic)
- **EXECUTION**: Task tool sub-agents, parallel
- **STORAGE**: Filesystem Markdown, no DB
- **FOLDER MODEL**: `debates/[YYYY-MM-DD-NNN-slug]/`
- **VERIFICATION**: Web search, Tier A/B/C/D hierarchy
- **NGÔN NGỮ**: Bilingual EN/VI, auto-detect
- **TESTING**: AutoTest (Karpathy-style loops)
- **VERSIONING**: `.baseline` snapshots + rollback
- **INPUT UX**: Direct-type in Cowork, no file edit

## Pipeline — 11 Steps (Automation-by-default)

*Chạy liền mạch trong 1 response*

1.  **Moderator**: Frame motion + definitions + shared premises
2.  **Persona Picker**: Assign 2 personas, framework distance ≥6
3.  **Debater PRO**: Argue FOR in persona voice, 4 rounds
4.  **Debater CON**: Argue AGAINST in distinct persona
5.  **Referee**: 8 procedural checks post-Rebuttal
6.  **FactCheck**: Independent web search + Tier audit
7.  **Judge U**: Utilitarian framework ballot (blind)
8.  **Judge R**: Rights-based framework ballot (blind)
9.  **Judge P**: Pragmatic framework ballot (blind)
10. **Meta-Judge**: Aggregate + reveal + Synthesis Part VII

## Quality Safeguards

- **Persona diversity**: framework distance ≥6
- **Parallel independence**: (R00 critical)
- **Referee procedural checks**: max 1 revision
- **Web search verification chain**: (Tier A/B/C/D)
- **True blind judging**: (sealed mapping, R14)
- **Ensemble 3 framework lenses**: + aggregate
- **Synthesis**: empirical vs normative split
- **AutoTest bounded loops**: + `.baseline` rollback
