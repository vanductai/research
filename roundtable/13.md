# 3. Debater (PRO + CON instances) ★ UNIFIED v2.0

**Argue assigned side với persona voice + framework, qua 4 vòng**

- **When**: Steps 2, 3, 4, 5 — parallel sub-agents per round
- **Input**: Per-round: `debate_brief.md` + `personas.md` + opponent's relevant prior file + `fallacy-catalog.md`
- **Output**: `{side}_{round}.md` — pro_opening, con_rebuttal, etc.

## Tính năng chính:

-   **Parameterized**: single agent file dùng cho cả PRO và CON, no duplication.
-   **Persona enforcement**: `PERSONA HEADER` mỗi file, voice + framework + evidence habits theo `personas.md`.
-   **Independence rule**: Round 1 CHỈ đọc brief; rounds 2-4 CHỈ đọc opponent file specific cho round đó. R00 violation = critical.
-   **Depth markers**: mỗi main argument phải có 2+ trong `[EVIDENCE]` `[MECHANISM]` `[ANTICIPATE]` `[IMPACT]`.
-   **Verified Facts**: `verified_facts[]` table mandatory cuối mỗi round file (V15.1 pattern).
-   **Self-Review**: `SELF_REVIEW` checklist trước khi save.
-   **Vòng tranh luận (4 rounds)**:
    -   **Opening**: (3 args + framework + pre-emption)
    -   **Rebuttal**: (steel-man + cross-exam)
    -   **Counter**: (answers + counter-rebuttal + extension)
    -   **Closing**: (summary + concession + final)
-   **Argument constraints**: NO new arguments in Counter/Closing (R12).
-   **Word count**: 800-1500/round, dưới 600 = R11.
