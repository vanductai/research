# Quality Safeguards - Debate Arena v2.0

|:---|:---|:---|

| **Procedural integrity** | Step 3.5 | Referee 8 checks + revision loop max 1 |
| **Fact verification** | Step 6 | Independent web search + Tier A/B/C/D + spot-check |
| **Blind judging** | Steps 7-8 | Random relabel + sealed mapping, R14 critical |
| **Single-judge bias** | Step 8 | 3 ensemble judges with distinct frameworks |
| **Synthesis quality** | Step 9 | 4 mandatory sub-sections, empirical vs normative split |
| **Iterative tuning** | AutoTest | Karpathy-style bounded loops with `.baseline` rollback |
